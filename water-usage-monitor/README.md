Water Usage Monitoring Web Application
======================================

This project is a full-stack water usage monitoring web application built with:

- Frontend: React (Vite) + Tailwind CSS + Chart.js
- Backend: Node.js + Express.js
- Database & Auth: Supabase (PostgreSQL + Supabase Auth)
- Deployment: Frontend on Vercel, Backend on Render, Database on Supabase

Project structure:

- `frontend/` – React + Vite + Tailwind SPA
- `backend/` – Express API server integrated with Supabase
- `database/` – Supabase SQL schema
