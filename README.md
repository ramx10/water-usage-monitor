# 💧 Water Usage Monitoring System

A full-stack web application that allows users to track daily water consumption, visualize usage trends, and monitor water conservation.

## 🚀 Features

* User Registration & Login
* Daily Water Usage Tracking
* Usage History
* Analytics Graph
* Smart Water Usage Alert
* Admin Dashboard
* Secure Authentication

## 🛠 Tech Stack

Frontend

* HTML / CSS / JavaScript
* Chart.js

Backend

* Node.js
* Express.js

Database

* Supabase (PostgreSQL)

Deployment

* Frontend: Vercel
* Backend: Render

## 📊 Dashboard Features

* Total Water Used Today
* Monthly Water Usage
* Average Daily Usage
* Usage Trend Graph
